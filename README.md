# Diverso WorkFlows

![Version](https://img.shields.io/badge/version-2.0-blue?style=flat-square)
![PWA](https://img.shields.io/badge/PWA-Ready-FF00AA?style=flat-square)
![License](https://img.shields.io/badge/license-MIT-green?style=flat-square)

**Seu cérebro externo visual. Leve, bonito e extremamente funcional.**

Organize ideias com post-its conectáveis, refine prompts com IA, gerencie projetos com Kanban + Fluxograma automático e transforme conceitos em realidade — tudo em uma interface neon moderna.

![Screenshot Principal](https://via.placeholder.com/1200x620.png?text= Screenshot+do+App+-+Diverso+WorkFlows)

## ✨ Principais Funcionalidades

- 💡 **Brainstorm Visual** — Post-its conectáveis com drag & drop e modo conexão inteligente
- ✍️ **Editor Múltiplo de Prompts** — Histórico de versões + geração e refinamento via IA
- 📋 **Kanban Inteligente** — Organizado automaticamente com ajuda da IA
- 🔀 **Fluxograma Interativo** — Etapas visuais com progresso automático
- 🐙 **Sync com GitHub + IA** — Analisa seu repositório e sugere fluxo, kanban e progresso
- 📚 **Biblioteca de Prompts** — Prompts salvos e reutilizáveis em qualquer projeto
- 🌙 **Modo Foco** — Ambiente sem distrações para concentração profunda
- 📊 **Relatório + Análise IA** — Avaliação inteligente do estado do projeto
- 💾 **Dados locais + Supabase** — Funciona offline e sincroniza na nuvem
- 📱 **PWA Completo** — Instale como aplicativo no desktop ou celular

## 🚀 Experimente Agora

**[Abrir Diverso WorkFlows →](https://diversoworkflows.vercel.app/app)**

## 🛠 Tecnologias

- **Frontend**: HTML, CSS, Vanilla JavaScript
- **IA**: OpenAI / GitHub Models (GPT-4o-mini e outros)
- **Banco de Dados**: Supabase (sincronização opcional)
- **Armazenamento**: LocalStorage + Supabase
- **Instalação**: Progressive Web App (PWA)

## Como Usar

1. Acesse [https://diversoworkflows.vercel.app/app](https://diversoworkflows.vercel.app/app)
2. Faça login com sua conta Google
3. Crie seu primeiro projeto e comece a trabalhar
